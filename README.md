# Stairs

**Stairs** is a top-down dungeon adventure game made with Python and Pygame.

---

## Requirements

* Python 3.10+
* [Pygame](https://www.pygame.org/) library

Install Pygame using pip:

```bash
pip install pygame
```

---

## Controls

| Key       | Action     |
| --------- | ---------- |
| **W**     | Move Up    |
| **A**     | Move Left  |
| **S**     | Move Down  |
| **D**     | Move Right |
| **Space** | Attack     |
| **Esc**   | Pause Game |
